# 07 — Technical

## Tech Stack
- **HTML5** — page structure
- **CSS3** — styling and layout
- **Vanilla JavaScript** — interactions and behaviour
- **No frameworks** — keeping it simple and lightweight

## Accessibility (WCAG AA)
- [ ] Colour contrast ratio minimum 4.5:1
- [ ] All images have alt text
- [ ] Keyboard navigable
- [ ] Focus states visible
- [ ] Semantic HTML used throughout
- [ ] Crisis numbers always visible

## Performance Goals
- Page load under 3 seconds on mobile
- Images compressed and optimised
- Minimal external dependencies
