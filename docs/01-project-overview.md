# 01 — Project Overview

## Project Name
SADAG Rebrand

## What is this project?
A rebranding and redesign of a South African mental health organisation's website as a social cause initiative, guided by a mentor.

## Goals
- [ ] Modernise the visual design and user experience
- [ ] Make mental health resources more accessible to all South Africans
- [ ] Improve mobile responsiveness
- [ ] Clearly surface crisis support information
- [ ] Build a site that feels warm, safe, and trustworthy

## Out of Scope
> List anything that is NOT part of this project
- Backend / CMS integration (TBD)
- Native mobile app
- Multi-language support (TBD)

## Target Audience
> Who is this site for?
- People experiencing a mental health crisis
- Friends/family supporting someone struggling
- People seeking ongoing support groups or therapy
- Healthcare professionals and advocates

## Success Criteria
> How will we know this project is done and done well?
- [ ] All pages designed and built
- [ ] Site is mobile-friendly
- [ ] Crisis helpline numbers are prominent on every page
- [ ] Deployed and accessible via a live URL
- [ ] Reviewed and approved by mentor
