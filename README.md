# Mentor Rebrand — Mental Health Site

A rebranding project for a South African mental health organisation, built as a static site (HTML/CSS/JS).

## 📌 Project Overview

This project is a social cause initiative to redesign and modernise a mental health support website for South African users, improving accessibility, design, and user experience.

## 🗂️ Project Structure

```
mentor-rebrand/
├── index.html          # Homepage
├── css/
│   ├── styles.css      # Main stylesheet
│   └── responsive.css  # Mobile & responsive styles
├── js/
│   └── main.js         # Main JavaScript
├── pages/
│   ├── about.html
│   ├── get-help.html
│   ├── support-groups.html
│   ├── resources.html
│   └── contact.html
└── assets/
    ├── images/
    └── icons/
```

## 🚀 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
